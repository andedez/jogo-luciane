Dicas rápidas

  • HTTPS é HTTP + camada de criptografia (TLS).
  • O cliente e o servidor trocam chaves e certificados antes de enviar dados.
  • Códigos 2xx indicam sucesso, 3xx redirecionamento, 4xx erro do cliente, 5xx erro do servidor.
